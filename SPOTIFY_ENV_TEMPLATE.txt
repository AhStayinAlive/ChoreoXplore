# Spotify API Configuration
# Get these values from https://developer.spotify.com/dashboard

VITE_SPOTIFY_CLIENT_ID=your_spotify_client_id_here
VITE_SPOTIFY_CLIENT_SECRET=your_spotify_client_secret_here
VITE_SPOTIFY_REDIRECT_URI=http://127.0.0.1:5137/callback

# Instructions:
# 1. Go to https://developer.spotify.com/dashboard
# 2. Create a new app
# 3. Copy your Client ID and Client Secret
# 4. Add redirect URI: http://127.0.0.1:5137/callback (for development)
# 5. Replace the values above with your actual credentials
# 6. For production, update the redirect URI to your domain
# 7. Note: Use 127.0.0.1 instead of localhost (Spotify requirement)
# 8. Note: Vite uses VITE_ prefix instead of REACT_APP_ prefix
# 9. Note: Port 5137 is configured in vite.config.js
